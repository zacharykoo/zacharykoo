Projects

Ecommerce Website Application

Backend API handling created with industrial approach where a handler is used to process each "GET", "POST" and "PUT" requests. SQLite is used for database management and GORM is used to handle the Query and creates the database and is written in Golang. Frontend user interface are created using Angular.

Distributed System

Designing and implementing distributed systems with system communications using paradigms, fault tolerance and coordination in Java. Communications and information transfer are ordered by timed system and ACKs are implemented to ensure data is sent correctly.

Supermarket Self-Checkout System

This project is designed to replicate a normal self-checkout system in a super market. Items can be scanned, bagged and price will be updated accordingly. Weight of the items are tracked in the bagging area to ensure items is correctly scanned. Payment methods includes Cash, Card and digital coins.

Nightowl-Studios - drawing game

This project is designed to replicate online real time drawing game. Websockets are implemented in order to get client-server both way communications. Game engine and game handlers were developed using Golang and Vue.
